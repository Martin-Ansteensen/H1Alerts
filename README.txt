# H1Alerts

H1Alerts is a surveillance-system for web-urls

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requests.

```bash
pip install requests
```

## Usage
Install cron

Example of a cronline that makes the script run every minute
```bash
* * * * * $(which python3) /home/copyleft/Desktop/H1Alerts/H1Alerts.py >> ~/cron.log 2>&1

```
